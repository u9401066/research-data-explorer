# Research Data Explorer Rules

Use the RDE MCP tools as the source of truth for real dataset analysis.

- Follow the 13-phase auditable pipeline and the workspace `AGENTS.md`.
- Do not bypass Phase 3, Phase 5, or Phase 6 confirmation gates.
- Do not run Phase 8 analysis without a locked plan.
- Record plan departures with `log_deviation()`.
- Cite generated artifacts in final answers.
- Keep outputs sanitized; do not expose raw sensitive paths or suspected PII.
