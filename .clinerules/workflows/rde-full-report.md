# RDE Full Report Workflow

When the user asks for a complete data analysis report:

1. Start with `get_pipeline_status()` or `init_project()`.
2. Run intake and schema tooling before interpreting variables.
3. Align research concepts to schema variables and wait for confirmation.
4. Generate a greedy analysis blueprint, then complete plan review and plan lock.
5. Run readiness checks before execution.
6. Execute planned analyses only; log deviations.
7. Collect results, assemble the report, audit it, then auto-improve.
8. Export final report or handoff package when requested.

Final responses should mention key artifact paths, audit grade/readiness, and remaining blockers.
